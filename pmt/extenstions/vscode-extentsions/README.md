# Project Management Tool Extension for VS Code

This VS Code extension provides project management capabilities directly within your development environment.

## Features

- Task and Todo management directly in VS Code
- Project initialization and configuration
- Team collaboration features
- GitHub integration for project repositories
- Progress tracking and goal setting

## Requirements

- Visual Studio Code version 1.60.0 or higher

## Extension Settings

This extension contributes the following settings:

* `projectJourney.openAIKey`: API key for OpenAI integration

## Getting Started

1. Install the extension
2. Open a project folder in VS Code
3. Use the PMT sidebar to access project management features
4. Initialize your project using the "PMT: Initialize Project" command

## Available Commands

* `PMT: Add Todo Item` - Add a new todo item to your list
* `PMT: Complete Todo Item` - Mark a todo item as complete
* `PMT: Create New Goal` - Set a new project goal
* `PMT: Track Progress` - View and update project progress
* `PMT: Generate Documentation` - Auto-generate project documentation
* `PMT: Create Task Automation` - Create automated task workflows
* `PMT: Continue Yesterday's Tasks` - Resume tasks from previous day
* `PMT: Initialize Project` - Set up project for tracking
